📝Task Manager App

A simple and interactive Task Manager built using HTML, CSS, and JavaScript.

🚀 Features
➕ Add tasks
🗑️ Delete tasks
✏️ Edit tasks
✅ Mark tasks as completed
🔍 Search tasks
🎯 Filter tasks (All / Completed / Pending)
🧹 Clear completed tasks
💾 Data stored using localStorage (persists after refresh)
📊 Live task counter

# What I Learned
Difference between UI (rendering) and data (state)
How to manage state using arrays and objects
Using localStorage to persist data
Event handling in JavaScript
DOM manipulation (rendering dynamic content)
